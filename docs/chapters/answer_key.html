解説

まずは導関数の基本公式を使う \[ \frac{d}{dx} \frac{1}{f(x)} = - \frac{f'(x)}{f(x)^2} \]

f(x) = \(1 + e^{-x}\) と置くと、 \[ f(x) = 1 + e^{-x}, \quad f'(x) = - e^{-x} \]

連鎖律を使って微分すると、

\[ \frac{d\sigma(x)}{dx} = \frac{d}{dx} \frac{1}{f(x)} = - \frac{f'(x)}{f(x)^2} = - \frac{-e^{-x}}{(1+e^{-x})^2} = \frac{e^{-x}}{(1+e^{-x})^2} \]

これをσ(x)を使った形に書き換える。

\[ \sigma(x) = \frac{1}{1+e^{-x}} \implies 1 - \sigma(x) = \frac{e^{-x}}{1 + e^{-x}} \]

よって、 \[ \frac{d}{dx}\sigma(x) = \frac{e^{-x}}{(1+e^{-x})^2} = \sigma(x)(1 - \sigma(x)) \]
